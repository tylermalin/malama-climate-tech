# Mālama Labs — Tech That Will Save the World

An interactive database of climate technologies with AI-powered insights from the Mālama Agent.

## Features

- **90+ Climate Technologies**: Comprehensive database covering renewable energy, carbon removal, sustainable agriculture, and more
- **AI Agent Integration**: Ask Mālama for deep dives on technologies or generate startup concepts
- **Interactive UI**: Expandable rows with detailed descriptions and curated resources
- **Search & Filter**: Find technologies by name, sector, or readiness level
- **Responsive Design**: Works seamlessly on desktop and mobile

## Technologies Used

- Pure HTML/CSS/JavaScript (no build step required)
- Google Generative AI SDK (Gemini 2.0 Flash)
- Modern CSS with glassmorphism and smooth animations

## Deployment

This project is optimized for Vercel deployment:

1. Push to GitHub
2. Import to Vercel
3. Deploy (no configuration needed)

## API Key

The application uses the Google Gemini API. To use your own API key:

1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create a new API key
3. Update `DEFAULT_API_KEY` in `table.html` (line ~983)

## Local Development

Simply open `table.html` in a modern web browser. No build process required!
